# OSM Location Picker
This is sample project generated by Gemini 3.0 Pro for testing purposes.

A framework-agnostic OpenStreetMap location picker library using Leaflet.

## Features

- Interactive map with draggable marker
- Real-time coordinate updates
- Built-in customizable coordinate overlay
- Framework agnostic (works with React, Angular, Vue, Vanilla JS, etc.)
- TypeScript support

## Usage

```typescript
import { LocationPicker } from 'osm-location-picker';
import 'osm-location-picker/dist/style.css'; // If styles are bundled, or just standard leaflet css

// Initialize the picker
const picker = new LocationPicker({
  targetId: 'map-container', // ID of the container element
  initialLat: 51.505,
  initialLng: -0.09,
  zoom: 13,
  enableOverlay: true, // Show the coordinate overlay
  onChange: (lat, lng) => {
    console.log('New location:', lat, lng);
  }
});

// Get current coordinates manually
const { lat, lng } = picker.getCoordinates();

// Cleanup
picker.destroy();
```

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `targetId` | `string` | **Required** | The ID of the HTML element to render the map in. |
| `initialLat` | `number` | `51.505` | Initial latitude. |
| `initialLng` | `number` | `-0.09` | Initial longitude. |
| `zoom` | `number` | `13` | Initial zoom level. |
| `enableOverlay` | `boolean` | `true` | Whether to show the coordinate overlay. |
| `overlayStyle` | `Partial<CSSStyleDeclaration>` | `undefined` | Custom styles for the overlay. |
| `onChange` | `(lat: number, lng: number) => void` | `undefined` | Callback for location changes. |
